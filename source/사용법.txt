***************  Packet Capture Program  ***************
버전 : 1.0.0
가동 가능 운영체제 : 리눅스
권장 운영체제 : 우분투
현재 로그 파일 저장 위치 : 프로그램 설치 위치


1) 소개
    - 해당 프로그램은 대학 과제 제출용 프로그램입니다.
    - 우분투에 환경에서 작성되어 테스트된 프로그램입니다.
    - raw socket을 활용하여 c언어로 작성된 패킷 분석 프로그램입니다.
    - 분석 가능한 프로토콜은 ICMP, DNS, HTTP, SSH 입니다.
    - 해당 프로그램에서 로그는 임시 저장 공간인 큐에 저장되어 분석 종료 후에 저장 희망 여부를 묻고 희망할 경우 원하는 이름의 텍스트 파일로 로그를 저장하는 방식입니다.
    - 임시 저장공간이 부족해도 분석을 종료하지 않습니다. (기본 설정은 10000줄입니다. 9번 메뉴를 눌러 항상 넉넉한 임시저장 공간을 설정해주세요.)
    - 임시 저장공간은 페이로드에 대해서는 한 줄씩 저장하지 않습니다. (화면 상에 보이는 줄보다 훨씬 적은 줄로 저장하기에 페이로드에 대한 걱정은 하지 않으셔도 됩니다.)
2)  기능
    1. 패킷 분석
        - Ethernet, IP, (ICMP/TCP/UDP), (DNS, HTTP, SSH) 순으로 분석 결과가 출력되고 저장됩니다.
        - 분석 도중에도 메뉴는 작동하기에 2, 3, 0번 메뉴는 이용할 수 있지만, 9번 메뉴는 심각한 오류를 초래할 수 있기에 접근이 불가능합니다.
    2. 분석 종료 및 결과 저장
        - 첫 실행 시 사용이 불가능합니다.
        - 실행 중인 패킷 분석이 있다면 종료 시키고 분석 결과 로그를 저장합니다.
        - 새로운 분석을 시작하기 전이나 최대 임시 저장 줄 설정을 하여 임시 저장 공간을 초기화하지 않았다면 전에 분석한 결과를 저장할 수 있습니다.
    3. 화면 비우기
        - 많은 출력로 화면이 복잡할 때 사용하실 수 있습니다.
        - 메뉴를 제외한 글을 지웁니다.
    9. 최대 임시 저장 줄 설정
        - 임시 저장 공간의 최대 크기를 설정합니다.
        - 사용할 시 임시로 저장된 분석결과는 초기화됩니다.
        - 기본 설정은 10000줄 입니다.
3) CMakeList.txt
    - cmake 파일로 cmake를 이용하시지 않는다면 다운받지 않아도 됩니다.
    - 해당 텍스트 파일을 어떻게 다룰지는 자유입니다. (따로 CMakeList.txt 파일을 작성하거나, cmake를 이용하지 않는다면 없어도 됩니다.)
